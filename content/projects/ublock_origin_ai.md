# uBlock Origin AI

A fork of uBlock Origin that experiments with using AI to assist in filter maintenance and tracker detection.

## Focus Areas

- **Filter Assistance**: Exploring how LLMs can help identify ad patterns and suggest filters for new sites.
- **Detection Patterns**: Testing behavior-based detection to complement existing static filter lists.
- **Efficiency**: Prioritizing the original project's focus on low CPU and memory usage.

## Technologies

- **Browser Extensions**: Works with standard Chromium and Firefox extension APIs.
- **Pattern Matching**: Uses local processing to identify ad structures.
- **Core Logic**: Built primarily with JavaScript.
