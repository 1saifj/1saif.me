# Salasto Backend

I lead the backend development at Salasto, focusing on building a stable and scalable environment for our services.

## Technical Details

- **Core Stack**: Built using Node.js, Express, and TypeScript for a maintainable and type-safe codebase.
- **Authentication**: Implemented PIN-based auth using SMS integration and OTP fallbacks to handle user identity reliably.
- **Data Management**: Use PostgreSQL with Drizzle ORM for schema management and Redis for session persistence.
- **Security**: Standardized on Argon2id for hashing and implemented common security headers and rate limiting.
- **Modular Design**: Structured the project to support multiple features independently, making it easier for the team to collaborate.

## Leadership

- Manage the backend team and coordinate technical decisions.
- Introduced standardized validation with Zod and automated documentation.
- Focus on mentoring and maintaining code quality across the team.
